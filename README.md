Docker Face Recognition WebService
===============================
author: Tobias Schoch

Overview
--------

A small web service running in a docker container that enables face recognition in images.


Change-Log
----------
##### 0.0.1
* initial version


Installation / Usage
--------------------

To install use pip:

    pip install <git-url>


Or clone the repo:

    git clone <git-url>
    python setup.py install
